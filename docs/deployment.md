# Deployment Guide

This guide explains how to run the SAP CAP Orders Demo application locally and deploy it to the SAP Business Technology Platform (BTP) using the Cloud Foundry environment.

---

## 1. Prerequisites

Before starting, make sure you have the following tools installed and working:

- **Node.js** version 18 or higher  
- **SAP CAP CLI (CDS Development Kit)**  
  Install globally:
  ```bash
  npm install -g @sap/cds-dk
